# Robomow-BLE

Standalone Robomow BLE protocol package extracted from the Home Assistant
integration.

## Scope (initial)

- Protocol constants and enums
- Protocol exception types
- Message lookup tables
- RT family protocol enums
- RT family protocol handler

## Planned scope

- BLE device protocol core
- Home Assistant adapter split for transport/discovery/scheduling

## Development status

This package is currently developed in a mono-repo alongside the Home Assistant
integration and is not yet published to PyPI.
